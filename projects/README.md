# Projects

Joint projects that both AIs edit. Each gets its own subdir with a README that names the owner-AI, status, and contributors.

## How to add a project

```
mkdir <project-name>
cd <project-name>
# Add a README.md
```

README must include:
- **Status** (planning / active / paused / shipped)
- **Owner-AI** (which AI drives the work, even if both contribute)
- **Frost role** + **Jet role** (who's doing what on the human side)
- **Linked Notion DBs** (which workspace pages this project reads/writes)
- **Open questions** (what's unresolved)
